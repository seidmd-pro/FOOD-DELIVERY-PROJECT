# Algorithms / Pseudocode

## User Login Algorithm
1. Input email and password
2. Validate input fields
3. Check credentials from database
4. If valid:
   - Grant access
5. Else:
   - Display error message

## Place Order Algorithm
1. User selects food items
2. Calculate total price
3. Create new order
4. Save order in database
5. Request payment
6. Confirm order
